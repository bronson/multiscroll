MultiScroll jQuery Plugin

Scroll a div by hovering, dragging, clicking, or mousewheel.

Dependencies: none (but the jQuery mousewheel plugin is bundled)

How to add to a Rails asset pipeline: gem 'multiscroll', :git => 'git://github.com/bronson/multiscroll.git'
